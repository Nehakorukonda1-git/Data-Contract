# Data Contract Documentation

Welcome to the LifeChurch Data Contract Documentation. This documentation provides detailed information about our data views, columns, and business logic.

## What is a Data Contract?

A data contract is a formal agreement that defines:
- **What** the data represents
- **Where** it comes from
- **Who** owns and maintains it
- **How** it should be used
- **Why** it matters to the organization

## How to Use This Documentation

### For Data Analysts
- Browse the **Column Reference** to understand individual fields
- Check **Data Quality Rules** to ensure proper usage
- Review **Example Values** to understand expected formats

### For Pastors & Ministry Leaders
- Use the **View Overviews** to understand what data is available
- Search for specific metrics or fields using the search bar
- Contact the data owner listed in each contract for questions

### For Data Engineers
- Reference **Assertions** for data quality checks
- Review **Related Fields** for join keys
- Check **Source** information for upstream dependencies
